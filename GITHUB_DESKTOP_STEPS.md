# Upload to GitHub using GitHub Desktop

## Quick Steps:

1. **GitHub Desktop should now be open** - if not, launch it from Start Menu

2. **Sign in** to your GitHub account (if not already signed in)

3. **Add your repository:**
   - Click **"File"** â†’ **"Add Local Repository"**
   - Click **"Choose..."** button
   - Navigate to: `C:\Users\Titus\Documents\Magnolia App`
   - Click **"Add Repository"**

4. **Review the changes:**
   - You'll see all your files listed
   - Add a commit message: "Initial commit: Magnolia Flight Portal"
   - Click **"Commit to main"** (button at bottom)

5. **Publish to GitHub:**
   - Click the **"Publish repository"** button (top right)
   - Repository name: `magnolia-flight-portal`
   - Choose **Private** or **Public**
   - **Uncheck** "Keep this code private" if you want it public
   - Click **"Publish Repository"**
